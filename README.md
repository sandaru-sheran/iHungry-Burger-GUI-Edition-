# iHungry Burger (GUI Edition) 🍔

A desktop Java application built with Swing to manage fast-food burger shop operations. This project demonstrates a strong understanding of Object-Oriented Programming (OOP) principles and basic MVC (Model-View-Controller) architecture.

## 🚀 Features

* **Interactive GUI:** User-friendly forms for placing, updating, and searching orders.
* **Order Management:** Create distinct `Burger` objects and store them using custom data structures (`BurgerList`).
* **Real-time Updates:** Modify order statuses (Preparing, Delivered, Canceled) and quantities via a dedicated controller.
* **Customer Lookup:** Search functionalities optimized through structured data rather than parallel arrays.

## 🛠️ Technical Implementation

* **Language:** Java (Swing/AWT for UI)
* **Architecture:** * **Model:** `Burger.java` acts as the data entity.
  * **View:** Various `.form` and `.java` files handling the Swing interface.
  * **Controller:** `BurgerController.java` manages the business logic and acts as the bridge between the UI and the data.
* **Key Concepts:** Object-Oriented Design, encapsulation, custom collections (`BurgerList`), and event-driven programming.
